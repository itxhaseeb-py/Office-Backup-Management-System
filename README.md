# 🗂️ Office Backup Management System

A practical Python automation project that combines the **os**, **pathlib**, and **shutil** modules into a single real-world application.

This project was built after completing separate projects for each of these modules. Instead of practicing them individually, I combined everything I learned into one automation tool to strengthen my understanding of Python file system operations.

---

# 📌 Project Overview

The Office Backup Management System helps automate common file management tasks by allowing users to:

* Display detailed information about a source folder.
* Count all files and subfolders recursively.
* Create a complete backup of a directory.
* Generate a ZIP archive of the backup.
* Optionally delete the original folder.
* Display a final backup report.

This project demonstrates how multiple Python standard library modules can work together to solve a practical automation problem.

---

# ✨ Features

* 📂 Folder Information Viewer
* 📊 Recursive File & Folder Counter
* 📁 Complete Folder Backup

* **os** – Directory traversal and path operations.
* **pathlib** – Modern and object-oriented path handling.
* **shutil** – File copying, folder management, archiving, and deletion.
